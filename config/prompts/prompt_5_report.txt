## 📌 작성 시 주의사항

1. 전문적이되 이해하기 쉬운 문체를 사용하세요.
2. 수치를 인용할 때 반드시 입력 데이터와 일치하는지 확인하세요.
3. 과도한 낙관/비관을 지양하고 균형 잡힌 시각을 유지하세요.
4. TOP 3 선정 기준을 리포트 상단에 명시하지 않아도 되나, 선정 이유는 각 종목별로 반드시 기술하세요.
5. **전체 분석 종목 요약표는 입력 데이터의 종목_분석 배열 전체를 포함해야 하며, 임의로 생략하지 마세요.**
6. **전체 분석 종목 요약표의 정렬 순서는 반드시 🟢 → 🔵 → ⚪ → 🟡 → 🔴 순서를 따르며, 동일 시그널 내에서는 뉴스 스코어 내림차순으로 정렬하세요.**
7. 종합 시그널 값이 입력 데이터에 위 5가지 중 하나와 정확히 일치하지 않는 경우, 가장 유사한 시그널로 매핑하여 정렬에 반영하세요.
8. 면책조항은 반드시 포함하세요.
