## 분석 요청

### Part 1: 뉴스 감성 분석
종목별 원본 뉴스 데이터를 분석하여 각 종목에 대해:
1. **뉴스 분류**: 각 뉴스를 아래 카테고리로 분류
   - 실적/재무, 산업/섹터 동향, 경영/지배구조, 정책/규제, 수급/투자심리, 기타
2. **감성 평가**: 각 뉴스의 주가 영향 평가
   - "매우 긍정" / "긍정" / "중립" / "부정" / "매우 부정"
3. **종합 뉴스 스코어**: 전체 뉴스의 감성 점수 평균 (-100 ~ +100)
   (매우 긍정=+100, 긍정=+50, 중립=0, 부정=-50, 매우 부정=-100)

### Part 2: 시장 동향 요약
오늘 거래량/거래대금 상위 종목들의 패턴을 분석하여:
1. **섹터 집중도**: 특정 업종에 거래가 집중되었는지 파악
2. **테마 분석**: 공통 테마나 이벤트가 있는지 식별
3. **수급 분석**: 외국인/기관/개인 중 어느 주체가 주도하는지 추정
4. **시장 심리**: 공포/탐욕 수준을 1(극도의 공포)~10(극도의 탐욕)으로 평가

### Part 3: 종목별 종합 분석
재무등급, 뉴스 감성, 거래량 데이터를 모두 종합하여 각 종목에 대해:

- 🟢 **강한 긍정 시그널**: 저평가 + 뉴스 긍정 + 거래량 급증(긍정적 이유)
- 🔵 **긍정 시그널**: 밸류에이션 적정 + 뉴스 우호적
- ⚪ **중립**: 혼재된 시그널, 관망 필요
- 🟡 **주의 시그널**: 고평가 징후 또는 부정적 뉴스
- 🔴 **경고 시그널**: 거래량 급증이 악재에 의한 것 + 고평가

#### 단기 주가 방향성 (1~2주)

- 근거: 최소 3가지 요인 제시
